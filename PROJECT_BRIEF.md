# 프로젝트 개요

QZ Briefing은 키움 OpenAPI+를 기반으로 국내 증시 브리핑을 제공하는 전용 프로그램이다. 자동매매나 주문 실행 기능은 포함하지 않는다.

## 핵심 기능

- 아침에 사용자가 영웅문에 로그인해 둔 상태에서 프로그램이 실행된다.
- 장전 브리핑에는 전일 장마감 분석과 최신 장전 데이터를 함께 반영한다.
- 오전 10시에는 현재 데이터를 전부 새로 수집하고 전일 마감 및 장전 데이터까지 포함해 최종 종합판단한다.
- 코스피 주도주 10개와 코스닥 주도주 10개를 선정한다.
- 코스피·코스닥 전체에서 바닥확인 및 반등초기 후보 5~10개를 선정한다.
- 보유종목마다 바닥확인, 물타기 대기·조건충족, 불타기 대기·조건충족, 관망, 위험축소 의견을 매일 제공한다.
- 물타기와 불타기는 자동주문하지 않고 조건과 기준가격만 제시한다.
- 각 지표마다 간결한 해석과 주의사항을 제공한다.
- 모든 브리핑에 데이터 기준시각과 데이터 누락 여부를 표시한다.
