# 요구사항

## 실행 환경

- Windows PowerShell 네이티브 환경에서 실행한다.
- WSL은 사용하지 않는다.
- Python 3.11 32-bit 가상환경 `.venv`를 사용한다.
- 키움 OpenAPI+와 영웅문 로그인 상태를 전제로 한다.

## 기능 요구사항

- 장전 브리핑은 전일 장마감 분석과 최신 장전 데이터를 함께 반영한다.
- 오전 10시 브리핑은 현재 데이터를 새로 수집하고 전일 마감 및 장전 데이터를 포함해 종합판단한다.
- 코스피 주도주 10개와 코스닥 주도주 10개를 선정한다.
- 코스피·코스닥 전체에서 바닥확인 및 반등초기 후보 5~10개를 선정한다.
- 보유종목별로 바닥확인, 물타기 대기·조건충족, 불타기 대기·조건충족, 관망, 위험축소 의견을 매일 제공한다.
- 물타기와 불타기는 조건과 기준가격만 제시한다.
- 각 지표에 현재 의미와 주의사항을 포함한 간결한 해석을 제공한다.
- 모든 브리핑에 데이터 기준시각과 데이터 누락 여부를 표시한다.

## 안전 및 데이터 요구사항

- 자동매매, 주문, 매수, 매도 기능을 구현하지 않는다.
- 계좌 정보는 읽기 전용으로만 사용한다.
- 실제 보유종목 데이터와 개인정보를 저장소에 커밋하지 않는다.
- 데이터 수집값, 계산값, 해석문을 분리한다.
- 사용자가 명시하지 않은 매매 규칙을 추가하지 않는다.

## 현재 단계 범위

- 프로젝트 문서와 기본 디렉터리 골격만 구성한다.
- 실제 분석 엔진과 키움 OpenAPI+ 연결 코드는 구현하지 않는다.
- 외부 Python 패키지는 추가하지 않는다.
